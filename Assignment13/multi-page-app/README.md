# 📘 Assignment (11/03/2026)

## Assignment Name

Multi-Page App

## Description

This project converts a React application into a **multi-page Single Page Application (SPA)** using **React Router**. It allows users to navigate between pages without reloading the browser.

---

## Technologies Used

* React.js
* JavaScript (ES6)
* React Router
* HTML5
* CSS3

---

## Features

* Navigation between multiple pages
* Client-side routing using React Router
* Dynamic page rendering without reload
* Simple and interactive user interface

---

## Pages

* Home Page
* About Page
* Contact Page

---

## How to Run

Navigate to the project folder

```
cd multi-page-app
```

Install dependencies

```
npm install
```

Start the development server

```
npm start
```

Open the application

```
http://localhost:3000
```

* Understanding **Single Page Applications (SPA)**
* Implementing **React Router**
* Navigation using **Link components**
