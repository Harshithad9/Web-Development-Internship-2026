# 📘 Assignment (05/03/2026)

## 🔎 Assignment Name: Component Hunt

### 📌 Description

The **Component Hunt** assignment focuses on understanding how modern websites are structured by breaking a popular website into smaller reusable UI components.

This exercise helps in learning **component-based design**, which is widely used in modern frontend frameworks like React.

---

## 🎯 Objective

* Analyze the layout of a popular website
* Identify reusable UI components
* Understand how large web pages are built from smaller parts
* Sketch the structural layout of the website

---

## 🧩 Identified Website Components

After analyzing the website, the following major components were identified:

* **Navbar / Header**

  * Logo
  * Navigation links
  * Search bar
  * Login / Profile icon

* **Hero Section**

  * Banner image
  * Headline text
  * Call-to-action button

* **Content Sections**

  * Cards / product blocks
  * Images and descriptions
  * Grid layout

* **Sidebar (if present)**

  * Filters
  * Categories
  * Quick links

* **Footer**

  * Contact information
  * Social media links
  * Useful links
  * Copyright information

---

## 📝 Sketch Structure

The website layout can be represented as:

```
Header / Navbar
│
├── Hero Section
│
├── Content Section
│   ├── Card 1
│   ├── Card 2
│   ├── Card 3
│
├── Sidebar (optional)
│
└── Footer
```
