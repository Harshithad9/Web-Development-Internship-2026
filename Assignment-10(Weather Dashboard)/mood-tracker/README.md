# 📘 Assignment (07/03/2026)

## 🌦️ Assignment Name: Weather Dashboard

### 📌 Description

The Weather Dashboard is a React application that fetches real-time weather data from a public API and displays it to the user.

The application handles loading and error states while fetching data, ensuring a smooth user experience.

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* CSS3
* OpenWeather API

---

## ✨ Features Implemented

* Search weather by city name
* Fetch data from a public API
* Display temperature and weather conditions
* Loading state while fetching data
* Error handling for invalid city names

---

## 🎯 Learning Outcomes

Through this assignment, I learned:

* How to fetch data using APIs in React
* Handling asynchronous operations
* Managing UI updates using React state
* Implementing loading and error states
* Building interactive dashboards
