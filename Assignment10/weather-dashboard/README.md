# 📘 Assignment (07/03/2026)

## 🌦️ Assignment Name: Weather Dashboard

### 📌 Description

The **Weather Dashboard** is a React application that fetches real-time weather data from a public API and displays it to the user.
The application allows users to search for a city and view its current weather conditions.

The project also handles **loading states while fetching data** and **error states for invalid city names or network issues**, ensuring a smooth user experience.

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* OpenWeather API
* Node.js
* VS Code

---

## ✨ Features Implemented

* Search weather by **city name**
* Fetch real-time weather data from a **public API**
* Display temperature, weather condition, and humidity
* **Loading indicator** while data is being fetched
* **Error message** for invalid city names or API issues
* Clean and interactive user interface

## ▶️ How to Run the Project

1. Navigate to the project folder

```
cd weather-dashboard
```

2. Install dependencies

```
npm install
```

3. Start the React development server

```
npm start
```

4. Open the application in your browser

```
http://localhost:3000
```
